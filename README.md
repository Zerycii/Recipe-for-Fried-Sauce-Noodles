## Recipe-for-Fried-Sauce-Noodles
家庭版炸酱面做法分享
# 准备食材
猪肉、木耳、干黄酱（我选则的是六必居干黄酱）、葱姜蒜、面粉、鸡蛋
# 食材处理
1.首先和面，在和面时可以加入一个鸡蛋和少许盐，盐可以使面条更筋道。将和好的面团放置盆中醒面15分钟，之后揉面5-10下。重复醒面揉面过程2-3次即可。
![面团](https://github.com/user-attachments/assets/2071097a-5ef0-45f0-9fe9-b212eabbab1c)
2.将猪肉切丁，大概筷子头到骰子块大小即可，猪肉选择五花肉或者其他部分都行。
![肉丁](https://github.com/user-attachments/assets/d3aa18f2-2e00-4660-b05f-1618cb28b6b2)
3.泡好的木耳切条，切的时候不要太细太小否则影响口感。
![木耳](https://github.com/user-attachments/assets/b1f01946-5618-48b8-a11d-473ca334433c)
4.将醒好的面团擀成面饼后再切成面条即可。
![面条](https://github.com/user-attachments/assets/a8aea99e-22bf-43fc-9229-c099493fcb07)
5.葱姜蒜切末。
# 炸酱
1.锅中放油，油量可稍微多一些，加入切好的肉丁翻炒至肉丁变色。
![炒肉](https://github.com/user-attachments/assets/3f3988de-5db8-46cd-902a-b0f8de28d4bd)
2.待肉丁全部变色以后加入干黄酱，我用的是六必居干黄酱（不是甜面酱），一般来说干黄酱的是苦咸口的，如果喜欢甜口可搭配适当甜面酱（7：3，需要更多甜度就增加甜面酱比例）但是主包个人不喜欢甜口所以没有加甜面酱。干黄酱在加入以前也可以先用水泄开，或者直接放入锅中炒再加水。切记炒酱过程中不要糊锅。主包用了一碗肉丁（普通吃饭用的碗）和半袋干黄酱。由于使用的酱比较多所以没有提前泄酱直接锅中炒。待黄酱在锅里翻炒泄开以后，加入切好的葱姜蒜和木耳。继续翻炒几下后加入适量的水。
![炒酱](https://github.com/user-attachments/assets/bdcb0ea9-d8d9-417a-a41a-2cb12ef7ff3d)
3.加水以后先用大火将水煮开，水开后转中小火熬酱15-20分钟左右，酱略微粘稠即可，熬好的酱是红棕色。熬制的过程中切记不要糊锅！
4.调料方面：由于黄酱本身是咸苦口，所以加入不到一勺盐即可（学校食堂最常见的铁勺，一般来说面条的卤会稍微咸一点）。生抽酱油一炒勺，一泵蚝油。正如主包所说，黄酱不仅咸还苦，所以可以加入白糖来中和苦味并且提鲜。白糖的用量是咸盐的2-3倍即可。
![熬酱](https://github.com/user-attachments/assets/690aa069-52a1-4466-9825-d892d5dca9c3)
# 成品展示
由于炸酱面本身只有肉和酱，所以本着营养均衡的原则可以加入一些配菜/菜码。在煮面过程中可以加入一些油菜，也可以搭配黄瓜、萝卜、西红柿等自己爱吃的蔬菜。主包还加入了一个煎蛋，也可以准备一些过年做好的酱牛肉、卤肉、丸子等。
![成品](https://github.com/user-attachments/assets/954b5081-0797-4230-9e6e-f6469b800ca8)
# 小结
炸酱面作为一种中国北方常见食物，尤其以老北京炸酱面出名。北京炸酱面讲究小碗干炸，油酱分离。但是在家庭版炸酱面制作过程中不易实现，因为油酱分离需要较多的油，如果油量较少则无法达到这一效果。一般家庭饮食追求清淡健康，所以在制作过程无法实现油酱分离也是正常现象。
