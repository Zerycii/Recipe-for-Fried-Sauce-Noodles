## Recipe-for-Fried-Sauce-Noodles
家庭版炸酱面做法分享
# 准备食材
猪肉、木耳、干黄酱（我选则的是六必居干黄酱）、葱姜蒜、面粉、鸡蛋
# 食材处理
1.首先和面，在和面时可以加入一个鸡蛋和少许盐，盐可以使面条更筋道。将和好的面团放置盆中醒面15分钟，之后揉面5-10下。重复醒面揉面过程2-3次即可。
![d8f53f57ba3f6e0522a82f33aefa7152](https://github.com/user-attachments/assets/0c553215-50b1-415b-a086-a3ea97f9ca0b)

2.将猪肉切丁，大概筷子头到骰子块大小即可，猪肉选择五花肉或者其他部分都行。
![e693ff9a06271da3afb7fd227675e3e9](https://github.com/user-attachments/assets/dbff4877-392b-49ae-94e1-b27ee26c28d3)
3.泡好的木耳切条，切的时候不要太细太小否则影响口感。
